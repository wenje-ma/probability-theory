# README

本项目为**概率论学习笔记**. 

## 参考资料

1.  Durrett, R. (2019). Probability: Theory and Examples (5th ed.). Cambridge University Press.
2.  孙振尧 (2025秋). 北京理工大学数学与统计学院 概率论 (双语) 课程板书 [EB/OL].

## 编排原则

笔记整体遵循**章节结构**编排, 内容收录严格执行以下规则: 

- **内容范围**: 仅覆盖课堂讲授内容, 教材包含但教师未提及的内容不纳入笔记. 
- **定理证明**: 以教材证法为核心蓝本, 优先保障证明过程的**简明直观性**. 

## 符号与定理说明

### 1. 符号规范

笔记符号体系以教材为基准, 仅对个别术语做通用化统一修正: 

- 教材中的 $\sigma$-field, 统一采用通用术语 **$\sigma$-代数** ($\sigma$-algebra) , 替代教材推荐的 **$\sigma$-域**. 

### 2. 定理名称修正

为对齐通用概率论体系, 对教材与通用体系的定理名称做如下统一调整: 

- 教材标注的 **Fubini 定理**, 对应通用体系的 Tonelli 定理, 笔记中统一记为 **Tonelli 定理**; Fubini 定理的标准内容将在后续章节单独阐述. 
- 教材标注的 **Chebyshev 不等式**, 对应通用体系的 Markov 不等式, 笔记中统一记为 **Markov 不等式**; Chebyshev 不等式的标准内容将在后续章节单独阐述. 

### 3. 课堂特有内容标记

教材未收录, 但课堂板书提及且需保留的定理, 定义等内容, 统一标注 **$^*$** 作为标识. 

## 章节调整说明

受课时限制, 课堂讲授积分定义时, 未采用教材及通用体系中 "从简单函数构造" 的经典方法, 而是直接以**乘积测度**为基础, 通过**下方图的几何意义**定义积分. 该方法可省略原教材 1.7 节 "乘积测度" 的独立讲解. 

基于此教学安排, 笔记章节内容做如下调整: 

1.  将原 1.7 节 "乘积测度" 的内容, 整合至 **1.4 节**, 节标题修改为 **1.4 积分 (乘积测度) **. 
1.  原 1.7 节标题调整为 **1.7 Fubini 定理**, 专门阐述 Fubini 定理的标准内容.
